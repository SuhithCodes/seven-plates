# **App Name**: MacroMetrics

## Core Features:

- Calorie Tracking: Display a calorie card showing burned vs remaining calories using a progress bar.
- Macro Tracking: Visualize macronutrient intake (protein, carbs, fat) with progress cards.
- Meal Suggestion: Suggest the next meal with preparation time using data from external recipe provider (API).
- Personalized Meal Planning: Recommend the number of meals per day and prep time based on user preference. LLM will act as a tool which recommends appropriate values based on user lifestyle.
- Dashboard Overview: Provide an overall dashboard view summarizing calorie and macro intake.

## Style Guidelines:

- Primary color: Soft olive green (#A2B582) to evoke health and nature.
- Background color: Desaturated, very dark olive green (#1A1C17).
- Accent color: Light mustard yellow (#E3D36A), for a touch of warmth and vibrancy.
- Body and headline font: 'Inter', sans-serif for a clean, modern look.
- Simple, minimalist icons for macronutrients and actions.
- Card-based layout for clear information grouping.
- Subtle transitions for progress bar updates.
